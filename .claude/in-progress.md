## 开发重点领域

### 论文处理
- Arxiv 论文精细翻译
- PDF 全文翻译（多线程）
- Latex 全文翻译/润色
- Latex 论文一键校对

### 代码功能
- 程序剖析（Python/C/C++/Java/Lua/...）
- 批量注释生成
- 代码解释

### LLM 集成
- OpenAI API (GPT-3.5, GPT-4)
- 国内模型（通义千问、智谱 GLM、文心一言、讯飞星火）
- 开源模型（LLaMA2、ChatGLM2、DeepseekCoder）
- 多 API-key 轮换

### 插件开发
- 插件放在 `crazy_functions/` 目录
- 支持热更新
- 参考 `core_functional.py` 和 `crazy_functional.py`

### UI 主题
- 默认主题
- 暗色主题 (`/?__theme=dark`)
- 自定义主题 (`themes/`)
